# [Princeton x Maseno] African Next Voices Clean Dataset

This repository contains clean datasets from the [African Next Voices](https://huggingface.co/Anv-ke) multilingual corpus compiled by the MCAA consortium for all languages shown in the table below:

| Corpus   | Filename         |
| -------- | ---------------- |
| Kikuyu   | `corpus_kik.csv` |
| Kalenjin | `corpus_kln.csv` |
| Dholuo   | `corpus_luo.csv` |
| Maasai   | `corpus_mas.csv` |
| Somali   | `corpus_som.csv` |

The data in this repository is licenced exclusively for use at the Princeton Laboratory for Artificial Intelligence.
